## If you intend to change the styling

1. Make sure you have [Node.js](https://nodejs.org) installed.
2. Install dependencies:

    ```bash
    npm install
    ```

3. Run:

    ```bash
    npx @tailwindcss/cli -i ./import.css -o ./src/styles.css --watch
    ```
